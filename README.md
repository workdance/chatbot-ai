## chatbot-ai

这个是 chatbot 的 ai 应用层代码，利用 Python 完成相关的代码。


### 已支持的模型

#### 本地模型
一期为了快速调试，主要支持本地的模型，主要通过 Ollama 启动服务，目前支持以下模型：
+ llama2
+ gemma:7b
+ qwen:14b

#### 远程模型

通过 API 调用的方式，支持远程模型的 API 调用
+ openai
